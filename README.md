# Asynchronous File Reader Service

A Node.js program that reads multiple files asynchronously using async/await.

## Features
- Reads three text files (a.txt, b.txt, c.txt) asynchronously
- Displays each file's content
- Calculates total characters across all files

## How to Run
1. Clone this repository
2. Navigate to the project folder
3. Run: `node index.js`
